# Taller Evaluado 04 — Funciones Utilitarias

Proyecto de funciones básicas en Python con suite de tests unitarios usando Pytest.

## Funciones implementadas

| Función | Descripción |
|---|---|
| `add(a, b)` | Retorna la suma de dos números |
| `divide(a, b)` | Retorna la división de `a` entre `b` |
| `get_element(list_, index)` | Retorna el elemento en el índice dado de una lista |
| `convert_to_integer(value)` | Convierte un valor string o float a entero |

## Instalación

```bash
pip install pytest
```

## Tests

```bash
pytest
```

## Estructura

```
taller_evaluado_04/
├── functions.py       # Implementación de las funciones
└── test_functions.py  # Tests unitarios
```
